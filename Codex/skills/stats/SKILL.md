# Stats

Report metrics only when their source and calculation can be reproduced. Include scope, time range, command or data source, value, and limitations.

Useful metrics include test counts, pass rate, build duration, changed files and lines, task completion, defect counts, and release-gate status. Label estimates as estimates. Never invent productivity, quality, or performance gains from incomplete evidence.
